Introduction

This page is a product of the Tech Team Mobile Applicaiton Development for the Capstone Class at The Ohio State University. Below is a general outline of how to develop a mobile application on both Android and iOS devices. We discuss different sensors both Apple and Android phones carry as well as the source code to interact with them.

Overview

This application is a template to show off what sensors mobile devices have access to. It is also a tool for other teams to use in order to improve their Capstone project. All code avaliable here is open-sourced and free to use for your own purpose. If you have any questions feel free to contact the developers, whose email is located in the Referance section. Enjoy!
